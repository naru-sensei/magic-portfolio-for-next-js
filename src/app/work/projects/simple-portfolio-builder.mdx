# 幾何学の魅力 - 図形で考える数学の世界

幾何学は、最も視覚的で直感的に理解できる数学の分野です。古代ギリシャから現代まで、人類の知的探求の中心にあり続けてきました。この講座では、幾何学の基礎から応用まで、その魅力を余すところなくお伝えします。

## 講座の特徴

この幾何学講座では、以下のような特徴があります：

- 豊富な図解と視覚的な説明で、抽象的な概念も理解しやすい
- 古典的なユークリッド幾何学から現代の微分幾何学まで幅広く紹介
- 数学史における幾何学の重要性と発展の歴史も解説
- 実生活や科学技術における幾何学の応用例も紹介

## 講座内容

### 第1回：ユークリッド幾何学の基礎
- 点、線、面の基本概念
- 三角形と円の性質
- 合同と相似
- 作図問題と証明

### 第2回：座標幾何学
- デカルト座標系
- 直線と円の方程式
- 二次曲線（放物線、楕円、双曲線）
- ベクトルと幾何学

### 第3回：三角法と計量幾何学
- 三角比と三角関数
- 三角形の解法
- 球面三角法
- 計量空間の概念

### 第4回：非ユークリッド幾何学
- 平行線公理の歴史
- 球面幾何学
- 双曲幾何学
- リーマン幾何学入門

### 第5回：現代幾何学と応用
- 微分幾何学の基礎
- トポロジーの考え方
- コンピュータグラフィックスと幾何学
- 物理学における幾何学（一般相対性理論など）
