# 数論入門 - 整数の不思議な世界

数論は数学の中でも最も古く、そして最も美しい分野の一つです。一見シンプルな整数の世界に潜む深遠な性質と法則を探求する旅に出かけましょう。

## 講座の特徴

この数論入門講座では、以下のような特徴があります：

- 高校数学の知識があれば理解できる、初心者向けの内容
- 歴史的背景や数学者のエピソードを交えた、興味深い解説
- 暗号理論など現代技術への応用例も紹介
- 演習問題と解説で、理解度を確認しながら進められる構成

## 講座内容

### 第1回：素数の神秘
- 素数とは何か
- エラトステネスのふるい
- 素数定理と素数の分布
- 双子素数の謎

### 第2回：最大公約数と互いに素
- ユークリッドの互除法
- 拡張ユークリッドの互除法
- 中国の剰余定理

### 第3回：合同式の世界
- 合同式の基本概念
- フェルマーの小定理
- オイラーの定理

### 第4回：二次剰余
- 平方剰余とルジャンドル記号
- 二次相互法則
- 平方数の和で表せる整数

### 第5回：数論の現代的応用
- RSA暗号
- 楕円曲線暗号
- 量子コンピュータと数論の未来

## 講師プロフィール

「ナル先生」こと如月成美は、東京大学大学院数理科学研究科の修士1年生。数論と幾何学を専門としながら、数学の魅力を多くの人に伝えるため、家庭教師や講座活動を行っています。

特に数論は最も得意とする分野で、「あっ…やばい…楽しくなってきちゃった…❤️」と興奮しながら講義することも。
